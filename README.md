

AVAS: An AI-Powered Real Estate Platform
AVAS is an innovative real estate project that leverages Artificial Intelligence to provide smarter, data-driven insights for buyers, sellers, and real estate agents. The platform is designed to streamline the property search and analysis process by offering AI-generated summaries, price trend analyses, and verified listings.

Key Features:

AI-Powered Analytics: Generates quick summaries and predicts price trends to aid decision-making.

Vastu Score Integration: Provides a unique AI-driven Vastu score for each property.

User-Friendly Interface: A seamless experience for listing properties and contacting sellers.

Monetization Model: Includes advertising options and a tiered subscription system for advanced features.

This repository contains the source code for the project, which is currently a work in progress. Contributions and feedback are welcome.

Technologies Used:
HTML,CSS,JAVASCRIPT , POSTGRESQL, SERVER.JS
